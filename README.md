# Textor
Textor is a Rails Engine to create TeX documents structure within Rails apps.

## Installation
Add this line to your application's Gemfile:

```ruby
gem "textor", :github => "nu12/textor"
```

And then execute:
```bash
$ bundle
```

## Usage
Run `$ rails db:migrate` to have the lastest migration.
